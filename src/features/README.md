# Features Layer

Слой для бизнес-функциональности и пользовательских действий.

## Структура фичи:
```
feature-name/
├── ui/           # UI компоненты фичи
├── model/        # Состояние и бизнес-логика
├── api/          # API запросы фичи
├── lib/          # Вспомогательные функции
└── index.ts      # Public API
```

## Примеры фич:
- auth (авторизация)
- add-to-cart (добавление в корзину)
- post-comment (комментирование)
- toggle-theme (переключение темы)
