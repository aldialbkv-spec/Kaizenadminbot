# Entities Layer

Слой для бизнес-сущностей предметной области.

## Структура сущности:
```
entity-name/
├── ui/           # UI представление сущности
├── model/        # Типы и модели данных
├── api/          # API для работы с сущностью
├── lib/          # Вспомогательные функции
└── index.ts      # Public API
```

## Примеры сущностей:
- user (пользователь)
- product (продукт)
- order (заказ)
- article (статья)
